# Rust Concept Practice

A hands-on Rust practice repository based on **The Rust Programming Language (Rust Book)**.  
This project is focused on building strong foundations across almost all important chapters required to become a professional Rust developer.

## Goal

- Practice core Rust concepts through small, focused examples
- Build practical intuition for ownership, borrowing, lifetimes, error handling, modules, collections, traits, and concurrency
- Maintain a chapter-by-chapter learning journey that can be revisited anytime

## Current Structure

- `src/main.rs` – default entry point
- `src/bin/ownership.rs` – ownership-focused practice binary
- `Cargo.toml` – crate configuration

As the repository grows, additional chapter/topic binaries can be added under `src/bin/`.

## Prerequisites

- [Rust](https://www.rust-lang.org/tools/install) (stable toolchain recommended)
- Cargo (installed with Rust)

Check your installation:

```bash
rustc --version
cargo --version
